# Breakout Brick Breaker Game

A classic Breakout arcade game built with React, Next.js, and TypeScript.

## Features

- **Paddle Control**: Control the paddle with mouse movement or keyboard (arrow keys)
- **Ball Physics**: Realistic ball bouncing with physics simulation
- **Destructible Bricks**: Multiple rows of colorful bricks to break
- **Score Tracking**: Points awarded for breaking bricks
- **Lives System**: 3 lives to complete the game
- **Game Over & Win Conditions**: Clear all bricks to win, lose all lives to game over
- **Pause Functionality**: Press Space to pause/resume the game
- **Responsive Design**: Clean, modern UI with Tailwind CSS

## How to Play

1. Move your mouse left/right to control the paddle
2. Or use Left/Right arrow keys
3. Keep the ball bouncing and break all the bricks
4. Don't let the ball fall below the paddle!
5. Press Space to pause/resume
6. Press R to restart the game

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

## Game Controls

- **Mouse**: Move paddle
